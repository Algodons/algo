# Sample Project

This is a sample project to demonstrate the Algo Code Editor features.

## Features Demonstrated

- **JavaScript** file with syntax highlighting
- **Python** file with type hints
- Multiple programming languages support
- File organization

## Try These Features

1. Open files from the file explorer
2. Edit code with syntax highlighting
3. Use Ctrl+S to save
4. Open the integrated terminal
5. Search across files
6. Split the editor view
